# Constants

## ChainId

Enum that represents chain ids. See [ChainList](https://chainlist.org/) for a list of known chains and their IDs.
The list of built-in chain IDs in useDapp can be found [here](https://github.com/TrueFiEng/useDApp/blob/master/packages/core/src/constants/chainId.ts).

**Values:**

`Mainnet, Goerli, Kovan, Rinkeby, Ropsten, BSC, Cronos, xDai, Polygon, Moonriver, Moonbeam, Mumbai, OasisEmerald, Harmony, Theta, Palm, Fantom, Avalanche, Songbird, Velas`
